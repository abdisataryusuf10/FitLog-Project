# FitLog - Fitness Tracker

A modern, responsive fitness tracking application built with React, Vite, and Tailwind CSS.

## Features

- **Log Workouts**: Add exercises with sets, reps, and weights
- **Workout History**: View and filter past workouts by date
- **Progress Tracking**: Visual charts showing fitness progress
- **Exercise Library**: Browse exercises from WGER API
- **Search Functionality**: Search exercises by name or muscle group
- **Responsive Design**: Works on mobile, tablet, and desktop
- **Local Storage**: Workouts are saved locally in your browser

## Technologies Used

- React 18
- Vite (Build tool)
- Tailwind CSS (Styling)
- React Router (Navigation)
- Recharts (Data visualization)
- date-fns (Date manipulation)
- WGER API (Exercise data)
